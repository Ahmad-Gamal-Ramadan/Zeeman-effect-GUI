## ⚛️ Zeeman Effect Simulator (GUI)

### 📌 Overview
A Python-based graphical user interface (GUI) for simulating the Zeeman effect in atomic systems.  
The tool supports Normal, Anomalous, and Hyperfine Zeeman splitting, allowing users to explore energy level behavior under external magnetic fields.

---

### ⚙️ Features
- Simulation of:
  - Normal Zeeman Effect  
  - Anomalous Zeeman Effect  
  - Hyperfine Zeeman Effect  
- Adjustable quantum numbers (L, S, I)  
- Hyperfine coupling constant (A) control  
- Magnetic field range control (B)  
- Multiple energy unit options (eV, meV, μeV)  
- Real-time visualization of energy levels  
- Optional labeling of quantum states  
- Export plots (PNG, PDF, SVG)  
- Export simulation data (CSV)  
- User-friendly interactive GUI
